How Open is OpenStack?

Open Source

  • 开源协议采用 Apache License 2.0
  • 企业可以拿来做成自己的产品
  • 参与者越来越多

Open Design

  • 非某公司拥有 （如Android)
  • 非某公司主导 （如CloudStack）
  • 基金会管理，共同参与决策
  • 想要更过话语权？请砸更多工程师

Open Devolopment

  • Public route map
  • Public code review
  • Free summit ticket

Open Community

  • Committee负责协调、选型和规模控制
  • 开发者友好：PTL-Core-Reviewer 的多级架构
  • 公司友好：无论商业还是开源方案，都可以找到支持
